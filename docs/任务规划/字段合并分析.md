# 简历表字段合并分析

## 字段重复性分析

### 重复字段（在Resume表中已存在）
- name, email, phone, address, birthDate, objective, profile
- createTime, updateTime

### 需要从ResumePersonalInfo添加的新字段
- avatar：头像URL
- jobTitle：职位名称
- selfEvaluation：自我评价
- interests：兴趣爱好（Resume表中是hobbies，可以统一）

### 需要从ResumeContact添加的新字段
- website：个人网站
- github：GitHub
- linkedin：LinkedIn

### 需要删除的字段（因为已单独表）
- education
- workExperience
- projects
- skills

## 合并方案
1. 在Resume表中添加avatar, jobTitle, selfEvaluation, interests, website, github, linkedin字段
2. 删除Resume表中的education, workExperience, projects, skills字段
3. 删除ResumePersonalInfo和ResumeContact表
4. 调整字段命名，统一命名规范
